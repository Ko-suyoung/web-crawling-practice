HTML정보 추출 - 선택한 태그에서 정보 추출¶

화면에 보이는 글 부분 : .text, ['속성명']인덱싱
화면에 보이지 않는 요소 : ...
ex) 하이퍼링크 : ['href'] (하이퍼링크는 모두 <ahref='URL주소'>형식)

> content = TAG.text # 태그에서 화면에 보이는 텍스트 부분만 가져옴
> attribute = TAG['속성명'] # 태그 내 속성값 가져오기

selenium.find_element...

selenium은 다양한 find_element 메소드들을 제공하며 입력요소의 차이존재
https://pythonblog.co.kr/coding/22/ 이 사이트가 잘 정리해 논 것을 가져옴

단일 element 복수 elements 설명
find_element_by_id - 태그의 id값으로 추출
find_element_by_name find_elements_by_name 태그의 name값으로 추출
find_element_by_xpath find_elements_by_xpath 태그의 경로로 추출
find_element_by_link_text find_elements_by_link_text a태그 텍스트값으로 추출
find_element_by_partial_link_text find_elements_by_partial_link_text a태그의 자식 텍스트값으로 추출 (..?)
find_element_by_tag_name find_elements_by_tag_name 태그 이름으로 추출
find_element_by_class_name find_elements_by_class_name 태그의 클래스명으로 추출
find_element_by_css_selector find_elements_by_css_selector css선택자로 추출

(beautifulsoup의 select메소드와 동일)
  • 단일element와 복수 elements는 추출 결과가 다름
    단일은 첫번째 element만 추출
    복수는 입력요소에 해당하는 모든 elements를 가져와 list로 저장
